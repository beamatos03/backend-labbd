# ✨Backend Node.js integrado ao Mongodb
>> Projeto criado na aula de Laboratório de Banco de Dados

## 📦Packages Utilizados
```
npm i express
npm i mongodb@4.2
npm i dotenv
npm i nodemon --dev
npm i express-validator
npm i bcryptjs
npm i jsonwebtoken
npm i cors
```
